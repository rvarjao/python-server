# Aula 03 --- Criando um servidor web

## Objetivo

Transformar o sistema em um servidor acessível pelo navegador.

## Conceitos

-   servidor web
-   rota
-   requisição e resposta
-   Flask

## Execução

Instalar Flask: pip install flask

Executar: python app.py

Abrir no navegador: http://localhost:5000
