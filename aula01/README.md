# Aula 01 --- Criando um sistema no console

## Objetivo da aula

Criar um pequeno sistema em Python apenas no terminal para entender a
lógica básica de um backend.

## Conceitos abordados

-   input e print
-   listas
-   estruturas de repetição
-   estruturas condicionais

## Sistema desenvolvido

Cadastro simples de alunos com: 1. Cadastrar 2. Listar 3. Sair

## Execução

python app.py

## Observação

Os dados ficam apenas em memória e desaparecem quando o programa fecha.
Na próxima aula resolveremos isso usando banco de dados.
